# Guía de Imágenes para UMATCH

Esta carpeta contiene todas las imágenes necesarias para la aplicación UMATCH.

## Estructura de Carpetas

### `/csu` - Fotografías del Centro de Servicios Universitarios
Imágenes del CSU de la UNAB para mostrar las instalaciones deportivas.

**Imágenes necesarias**:
- `hero-csu.jpg` - Imagen panorámica principal del CSU (1920x600px recomendado)
- `cancha-futbol.jpg` - Vista de las canchas de fútbol (800x600px)
- `cancha-basketball.jpg` - Vista de las canchas de basketball (800x600px)
- `cancha-tenis.jpg` - Vista de las canchas de tenis (800x600px)
- `cancha-padel.jpg` - Vista de las canchas de pádel (800x600px)
- `cancha-volleyball.jpg` - Vista de las canchas de volleyball (800x600px)
- `zona-billar.jpg` - Vista de la zona de billar (800x600px)
- `zona-pingpong.jpg` - Vista de las mesas de ping pong (800x600px)

### `/sports` - Iconos e Imágenes de Deportes
Imágenes específicas para cada deporte (opcional si se usan iconos SVG).

**Formato recomendado**: PNG con fondo transparente o JPG
**Dimensiones**: 400x400px para iconos cuadrados

### `/placeholders` - Imágenes Temporales
Imágenes placeholder mientras se consiguen las fotos reales.

## Recomendaciones de Fotografía

1. **Iluminación**: Tomar fotos durante el día con buena luz natural
2. **Resolución**: Mínimo 1920px de ancho para imágenes hero
3. **Formato**: JPG para fotos, PNG para logos/iconos con transparencia
4. **Optimización**: Comprimir imágenes antes de subirlas (usar TinyPNG o similar)
5. **Nombre de archivos**: Usar kebab-case (palabras-separadas-por-guiones)

## Próximos Pasos

Una vez tengas las fotografías:
1. Optimiza las imágenes para web
2. Renómbralas según la convención arriba
3. Colócalas en sus respectivas carpetas
4. Las referencias en el código ya están preparadas para recibirlas
